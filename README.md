# Python Statistical Analysis

This project demonstrates numerical computing and statistical analysis using Python.

## Project Overview

The project contains two independent analytical tasks.

### Task 1: Gaussian Elimination

A system of linear equations is read from an input file and solved using a manually implemented Gaussian elimination algorithm.

### Task 2: Boston Housing Statistics

The project analyzes the MEDV variable from the Boston Housing dataset.

Statistics calculated include:

- Maximum
- Minimum
- Range
- Mean
- Mode
- Variance
- Standard Deviation
- Percentiles

## Skills Demonstrated

- Python
- File I/O
- Linear algebra
- Gaussian elimination
- Statistical analysis
- Algorithm design
- Data processing

## Files


- `statistical_analysis.py` — Main program containing the statistical analysis and Gaussian elimination algorithms.
- `BostonHousing.csv` — Dataset used for statistical analysis.
- `input.txt` — Input file containing the system of linear equations.
